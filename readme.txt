Widgets for use with TNG genealogy software. Requires the TNG Wordpress Plugin.

== Description ==

Adds widgets for use on Wordpress sites integrated with The Next Generation (TNG) genealogy software. Requires [TNG Wordpress Integration plugin] (http://wordpress.org/extend/plugins/tng-wordpress-plugin/ "Official TNG Wordpress Integration") (aka TNG WordPress Integration).

The widgets are based on the built and user contributed mods for TNG.

Adds the following widgets:

*   TNG Menu - menu of links to TNG pages.
*   TNG Search Names - basic name and surname search box.
*   TNG Surnames Cloud - text cloud of top surnames in TNG database.
*   TNG Profile Box - displays basic individual profile from TNG ID.
*   TNG Top Surnames - list of top surnames in TNG database.
*   TNG Births - List of births from current date.
*   TNG Anniversaries - List of anniversaries from current date.
*   TNG Deaths - List of deaths from current date.

== Installation ==

This plugin will not activate if TNG Wordpress Plugin is not installed.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place a widget in a chosen widget area.
